## Floor
Find the nearest integer less than or equal to the parameter

```glsl
float floor(float x)  
vec2 floor(vec2 x)  
vec3 floor(vec3 x)  
vec4 floor(vec4 x)
```

### Parameters
```x``` Specify the value to evaluate.

### Description
```sign()``` returns a value equal to the nearest integer that is less than or equal to x.
